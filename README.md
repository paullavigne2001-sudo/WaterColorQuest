# Water Sort Puzzle — version allégée pour GitHub

Version regroupée pour éviter un dépôt trop fractionné.

## Fichiers utiles

- `src/App.jsx` : le jeu complet dans un seul fichier
- `src/main.jsx` : point d'entrée React
- `package.json` : dépendances et scripts
- `vite.config.js` : configuration Vite
- `.devcontainer/devcontainer.json` : démarrage direct dans Codespaces

## Lancer en local

```bash
npm install
npm run dev
```

## Lancer dans Codespaces

1. Dézipper puis envoyer tout le contenu à la racine du dépôt GitHub.
2. Ouvrir le dépôt dans Codespaces.
3. Attendre la fin de `npm install`.
4. Lancer :

```bash
npm run dev
```

Le port 5173 s'ouvre automatiquement.
